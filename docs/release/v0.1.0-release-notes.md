## SkillCopilot 0.1.0 (Release Candidate)

Local-first desktop copilot for AI project handoff and Skill/Agent management on Windows 10/11.

### Highlights

- Dashboard: read-only `HANDOFF.md`, `AGENTS.md`, and Git status
- Skills: scan local `SKILL.md`, search, read-only detail, copy
- Agents: scan project Agent configs, search, read-only detail, copy prompts
- Workspace: native folder picker + remembered path (`skillcopilot.workspaceRoot`)
- UI languages: 简体中文 / English / 繁體中文

### Local-first

SkillCopilot does not upload project contents, does not call a model API, does not add telemetry or accounts, and does not introduce SQLite in this release. Scanners are read-only.

### Install

Download the Windows MSI or NSIS installer from this GitHub Release once it is published.

These installers are **not commercially code-signed**. Windows SmartScreen may show a warning. Prefer verifying the SHA-256 hashes below before installing.
